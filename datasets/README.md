# Shared Datasets

Input data for labs. Organized by format.

## Structure

```
datasets/
├── csv/
├── json/
└── parquet/
```

## Conventions

- Use lowercase, snake_case filenames (e.g. `customer_orders.csv`)
- Include a brief description in the lab README when a dataset is lab-specific
- Shared datasets used across multiple labs live here
- Lab-specific datasets live in `labs/<lab_name>/datasets/`

## Adding Data

Place files in the appropriate format subdirectory. Keep datasets small enough for local development. Large datasets should be generated by lab scripts or downloaded via documented setup steps.
